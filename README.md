# li-time

[li-time](https://www.npmjs.com/package/li-time)用来获取年、月、日、时、分、秒的时间。

# 安装

```
npm i li-time
```

# 例子

```js
const now = require('li-time');

// 当前时间
now().date 

// 年份
now().year

// 年月
now().month

// 年月日
now().day

// 年月日时分
now().minute

// 年月日时分秒
now().second
```
